# История изменений QR Bot Server

## v2.0 - 2025-11-09

### Обновление фильтра мата

**Что изменилось:**

1. **Добавлен транслит**
   - blyat, blya, blyad, bleat
   - hui, huy, huilo, huylo
   - pizda, pizd, pizdec, pizdets
   - suka, nahui, mudak, govno
   - И другие варианты

2. **Изменена логика фильтрации**
   - Раньше: мат заменялся звёздочками
   - Теперь: имена с матом **полностью отклоняются**
   - Пользователь видит красивую страницу ошибки

3. **Обновлен текст**
   - Было: "Мат фильтруется автоматически"
   - Стало: "Мат запрещён в именах"

4. **Расширены словари**
   - RUSSIAN_PROFANITY - русский мат
   - ENGLISH_PROFANITY - английский мат
   - TRANSLIT_PROFANITY - транслит мата
   - ALL_PROFANITY - объединенный список
